## Setup

```bash
$ npm install
```

## Run dev

```bash
$ npm run dev
```

## Run build

```bash
$ npm run build
```
Compiled code will be in `./server/public/dist/` folder.

## Run locally

```bash
$ npm start
```
Site will be shown at http://localhost:3000/
